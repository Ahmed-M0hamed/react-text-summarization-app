# text summarization
this real time text summarization app with react and hugging face 
we used hugging face pretrained model for the inference 



## how to run the app 
- clone the repo 
``bach 
git clone repo 
``` 
```bash 
cd repo 
cd backend 
uvicorn main:app --reload
```

- run the app 

```bash
cd frontend 
npm install 
npm start 
```
